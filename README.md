BrandForge is a state-aware branding intelligence engine designed to eliminate the strategic gap between abstract brand theory and high-fidelity visual execution. Developed for elite design environments, the platform orchestrates a **Sequential Intelligence Pipeline (S.I.P)** to ensure every design asset is architecturally anchored in psychological and strategic data.

---

## 🎯 Who is it for?
BrandForge is an **Enterprise-Grade Branding Intelligence Suite** designed for:
- **Brand Strategists**: Who require a systematic, data-tethered bridge between abstract theory and visual execution.
- **Design Agencies**: Needing to eliminate the "Strategy-Visual Disconnect" in their internal handoffs.
- **Project Stakeholders**: Who demand high-fidelity, strategy-anchored brand manuals in real-time.

## 💡 What it does?
It transforms abstract brand DNA into a deterministic visual system. BrandForge orchestrates the entire branding lifecycle—from automated discovery and Jungian archetype mapping to the generation of WCAG-compliant design tokens and 1:1 professional PDF manuals.

## 🌟 Value Proposition
- **Strategy-Visual Parity**: Ensures that every visual asset is 100% anchored in strategic data.
- **Zero-Friction Handoff**: Professional-grade PDF reporting and industry-standard folder structures.
- **Technical Rigor**: Built on a non-distractive **Commander Console** framework with absolute technical focus.

---

## 🧩 The 5-Station Architecture

BrandForge is structured as a series of specialized stations, each with a dedicated **Interface (UI)** and **Intelligence (Logic)** layer.

1.  **Station 1: Brand Discovery (Extraction)**: Multi-step data ingestion via manual input or high-density Google Forms automation.
2.  **Station 2: Strategy Engine (Synthesis)**: Deep mapping of Triple-Archetype models and Maslow hierarchies with state-aware "Data Healer" middleware.
3.  **Station 3: Logo Assistant (Alchemy)**: Linguistic noun toolkits (50+ constructs) and Propositional Density (Pd) feedback loops.
4.  **Station 4: Design System (The Engine)**: Deterministic mapping of "vibe" markers to WCAG-compliant design tokens and typography.
5.  **Station 5: The Export Engine (Handoff)**: AI-synthesized markdown documentation and 1:1 parity PDF report generation.

---

## 🧬 The Platform Backbone: S.I.P
BrandForge utilizes a **Sequential Intelligence Pipeline (S.I.P)**—a state-aware data inheritance model. Every action in Station 1 (Discovery) flow-controls the weights for subsequent stations, ensuring that your Design System is a forensic reflection of your Brand Strategy.

> [!NOTE]
> The current architecture reflects the high-density technical benchmarks certified in **commit `36b05a6`**, featuring enhanced AI resilience and cross-station data persistence.

## ⚙️ Technical Stack & Operational Standards

- **Core**: React 19, TypeScript, Vite.
- **Styling**: Tailwind CSS / Vanilla CSS hybrid ("Commander Console" standard).
- **Intelligence**: Google Gemini (Pro/Flash), OpenAI (GPT-4o), Anthropic (Claude 3.5).
- **Persistence**: Firebase Firestore & Local Storage.
- **Operations**: GitHub Actions CI/CD and Docker Containerization.

## 🚀 Quick Start

1. **Clone & Install**:
   ```bash
   git clone https://github.com/gwatanavictorj/BrandForge.git
   npm install
   ```
2. **Configure Environment**:
   Initialize `.env` with your `GEMINI_API_KEY` (see [Env Reference](docs/security_and_environment.md)).
3. **Launch**:
   ```bash
   npm run dev
   ```
4. **Configure The Vault**:
   Access the **Settings Console** within the app to activate your preferred AI provider.
